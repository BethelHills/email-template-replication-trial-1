# HTML Email Template

This replicates the visual layout style but uses neutral branding.

HTML email template replication and Gmail deliverability testing using table-based layouts and inline CSS.

## Gmail Deliverability Adjustments

| Adjustment            | Reason                          |
| --------------------- | ------------------------------- |
| Inline CSS            | better email compatibility      |
| Table layout          | stable rendering across clients |
| Text to image balance | reduces spam scoring            |
| Neutral wording       | avoids spam trigger phrases     |
| Alt text for images   | improves deliverability         |
| Absolute URLs         | required by many email clients  |
